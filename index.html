Exploiting Speculative Execution for Memory Leakage

CVE-2017-5753 | Microarchitectural Security Assignment | 2025

Abstract

Spectre Variant 1 is a critical microarchitectural vulnerability that exploits speculative execution in modern processors to break memory isolation boundaries. By manipulating branch prediction and leveraging cache timing side channels, attackers can extract arbitrary data from protected memory regions.

This project presents a comprehensive analysis of the vulnerability, including its root cause in CPU microarchitecture, a practical proof-of-concept implementation, and an evaluation of current mitigation strategies. Our simulation successfully demonstrates the extraction of secret data from memory using only unprivileged code, highlighting the fundamental security challenges posed by performance-oriented processor design.
